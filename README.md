# 💬 NalanTamil Chat

A real-time group chat application built with React, Flask, Socket.io and MongoDB.

## Features
- User Signup and Login with JWT authentication
- Real-time group messaging using WebSockets (Socket.io)
- Message history saved to MongoDB
- Edit and delete your own messages
- Message reactions (👍❤️😂)
- Typing indicator
- Beautiful dark UI

## Tech Stack
- **Frontend:** React.js
- **Backend:** Python Flask + Flask-SocketIO
- **Database:** MongoDB
- **Auth:** JWT (JSON Web Token)
